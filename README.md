\# Portfolio Project



A professional personal portfolio website built using Django, HTML, CSS, Bootstrap, and Python.



This project showcases:

\- personal profile

\- projects

\- certificates

\- skills

\- contact information

\- media uploads



\---



\## Features



\- Responsive portfolio website

\- Django backend integration

\- Bootstrap-based modern UI

\- Certificate upload/display section

\- Profile image support

\- Static and media file handling

\- Admin panel support



\---



\## Technologies Used



\- Python

\- Django

\- HTML5

\- CSS3

\- Bootstrap

\- SQLite3

\## Installation Guide



\### 1. Clone Repository



```bash

git clone https://github.com/YOUR\_USERNAME/portfolio\_project.git

```



\### 2. Move Into Project Directory



```bash

cd portfolio\_project

```



\### 3. Create Virtual Environment



```bash

python -m venv venv

```



\### 4. Activate Virtual Environment



\#### Windows



```bash

venv\\Scripts\\activate

```



\---



\### 5. Install Dependencies



```bash

pip install -r requirements.txt

```



\---



\### 6. Run Django Server



```bash

python manage.py runserver

```
